# Flujo-de-Control-part1
Crear un programa en C++ que lea el sueldo de un empleado, calcule el ISR (Impuesto Sobre la Renta) y muestre en pantalla el sueldo y el descuento de ISR a ser descontado, de acuerdo con una escala salarial disponible en la plataforma de la DGII, si no aplica para descuento, mostrar un cero (0) o N/A.
